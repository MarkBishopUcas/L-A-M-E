L-A-M-E

L-A-M-E, also known as the LLM agent manager experiment, is a program designed to turn any LLM into an AI agent with direct access to your computer and the ability to perform complex actions to achieve a set goal.

LAME installation guide (Windows 11 /(untested 10)
1. Download the Python file
2. Download .bat File
3. (optional) Download icon.ico (sub in any image named icon.ico to replace the icon)
4. (optional) Download images (only required if you want LAME to use the Clade desktop app instead of an api)
5. Ensure all files share a file folder (DO NOT RUN IN YOUR DOWNLOADS DIRECTORY, CREATE A SEPARATE FILE FOLDER)
6. Run the .bat file
7. After running the .bat file, you will find the newly generated .exe file in the dist folder.
8. (optional) Drag and drop the image file folder into the same file path as the .exe folder.
9. You can now move the .exe file wherever you want (Recommended: create a shortcut to the .exe, and leave the file folder with the .exe inside)

Usage:

1. Open the .exe
2. Find and select which api's you have access to, and paste in the api key (WARNING, api keys do not save automatically (yet))
3. (optional) Tick the box allowing the Claude desktop app to work as a fallback
4. all api keys you enter will work in descending order. If one fails, it will automatically switch to the next api, pasting all previous information, and automatically continue without a hitch.
4. save and continue 
5. Done! You can now give instructions to your agent, and it will automatically perform them

Current Features
1. This project can give LLMs control over your keyboard and mouse to become personal AI assistants. Give them a task, and it will be completed 
2. can store multiple LLM keys to use as a backup, in case the current LLM runs out of tokens, goes offline, or fails
3. (Planned) ability to use your own locally run LLM
4. (Planned) ability to store LLM keys locally on the program so you dont have to re-type them every time.
